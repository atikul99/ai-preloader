== Description ==

AI Preloader animation makes a good initial impression on the user and gives the webpage a more professional look. It shows the user your commitment to delivering a high-quality experience and that you don’t leave any stones unturned.

### Features

* Allow showing preloader on a specific page.
* Custom background color. Also can change the opacity.
* Pure CSS3 animation.
* Add a custom image (jpg, png, gif, svg).
* Light weight, fast.

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the AI Preloader plugin through the ‘Plugins’ menu in WordPress.
3. In the WordPress administration screen, navigate to AI Preloader, check the settings there.

== Frequently Asked Questions ==

= Does it work with any theme? =

Absolutely! AI Preloader will works with any theme.

= How can I change the settings of the preloader? =

Navigate to AI Preloader. Here you will find all the AI Preloader settings.
